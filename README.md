# Paper-code
Code for reproducing results in my paper on lithium resources
> *The implications of uncertainties on global lithium resources availability estimations*  
> Author(s) Dongping Zhou*, Steve Pye, Brunilde Verrier, Paul E.Dodds 

The main code is provided in [`analysis.py`](analysis.py).  
It includes functions for simulation filtering, depletion analysis, cost integration, and figure generation.
